# Diagramas de Modelagem

Este diretório reúne diagramas auxiliares de modelagem que apoiam a arquitetura proposta:

- **Modelagem de dados** — esquemas das tabelas DynamoDB e RDS.
- **Diagramas de sequência** — fluxos de eventos entre produtores, SNS, SQS e consumidores Lambda.
- **Modelagem de eventos** — contratos dos principais eventos de domínio (`pedido.criado`, `arquivo.enviado`, `usuario.cadastrado` etc.).

> Para a arquitetura geral da solução, consulte [`../arquitetura/`](../arquitetura/) e [`../../arquitetura/descricao-arquitetura.docx`](../../arquitetura/descricao-arquitetura.docx).
